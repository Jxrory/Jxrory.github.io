# 引入思源字体

## link 方法

```css
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Noto+Sans+SC:100,300,400,500,700,900">
```

在需要使用思源黑体的地方设置 CSS 样式:

```css
font-family: 'Noto Sans SC';
font-weight: 100;
```

将数据本地化 **(备份)**: [备份压缩包数据](/_media/fontend/fonts/notosanssc.zip)
