# Netcat 详细使用教程

## 一、Netcat简介

nc 的基本功能如下：

- telnet / 获取系统 banner 信息
- 传输文本信息
- 传输文件和目录
- 加密传输文件
- 端口扫描
- 远程控制 / 正方向 shell
- 流媒体服务器
- 远程克隆硬盘

## 二、Netcat常用参数：

- -l： 开启监听
- -p：指定端口
- -t： 以telnet形式应答
- -e：程序重定向
- -n：以数字形式表示ip
- -v：显示执行命令过程
- -z : 不进行交互，直接显示结果
- -u ：使用UDP协议传输
- -w : 设置超时时间
