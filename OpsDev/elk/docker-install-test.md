# 使用 docker 搭建 ELK 测试版

基于 `https://github.com/deviantony/docker-elk` 开源项目.

## 安装

### clone

```bash
git clone https://github.com/deviantony/docker-elk.git /app/docker-elk
```

### 启动验证

```bash
docker-compose up
```

在浏览器中访问: [http://localhost:9200](http://localhost:9200)

默认用户名: `elastic`

默认密码: `changeme`
